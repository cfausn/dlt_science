Implementation Overview

Basic Implementation

Add the following code to your website to implement the Hedera donation widget:

<!-- 1. Add a container for the widget -->
<div id="hedera-donation-widget"></div>

<!-- 2. Add the script and initialize the widget -->
<script src="embed.js"></script>
<script>
window.initHederaDonationWidget({
containerId: 'hedera-donation-widget',
receiverId: '0.0.5680094', // Your Hedera account ID
title: 'Support Our Project',
primaryColor: '#00a79d',
showFooter: true,
testnet: true // Set to false for mainnet
});
</script>

Note: For detailed implementation examples including customization options, please visit the individual demo pages.

This demo runs on the Hedera Testnet. No real funds are used.
