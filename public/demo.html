Standard Implementation

Basic widget with default settings:

Implementation Code:

<div id="hedera-donation-widget"></div>

<script src="embed.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    initHederaDonationWidget();
  });
</script>

Customized Implementation

Widget with custom title, colors and size:

Implementation Code:

<div id="custom-donation-widget" 
     data-hedera-widget 
     data-title="Support Our Project" 
     data-primary-color="#3182ce"
     data-max-width="100%"
     data-testnet="true">
</div>

<script src="embed.js"></script>
