Hedera Donation Widget

A simple, customizable widget for accepting HBAR and stablecoin donations on your website

Live Demo

Try out the widget below to see how it works:

Implementation

Add the following code to your website to implement the Hedera donation widget:

<!-- 1. Add a container for the widget -->
<div id="donation-widget"></div>

<!-- 2. Add the script and initialize the widget -->
<script src="embed.js"></script>
<script>
window.initHederaDonationWidget({
  containerId: 'donation-widget',
  receiverId: '0.0.5680094', // Your Hedera account ID
  title: 'Support Our Project',
  primaryColor: '#00a79d',
  showFooter: true,
  testnet: true // Set to false for mainnet
});
</script>

Features

Easy Integration

Simply include the script and add a container element to your webpage. The widget handles everything else.

Multiple Tokens

Accept donations in HBAR and USDC tokens, with the flexibility to add more in the future.

Wallet Integration

Seamless integration with HashPack wallet for secure and easy transactions on the Hedera network.

Customizable Design

Easily customize colors, titles, and other properties to match your website's branding.

Responsive Layout

Works beautifully on all devices from mobile phones to desktop computers.

Testnet Support

Develop and test with Hedera Testnet before deploying to mainnet for real transactions.

This demo runs on the Hedera Testnet. No real funds are used.
